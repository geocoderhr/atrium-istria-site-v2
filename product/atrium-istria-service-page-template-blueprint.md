# Atrium Istria Service Page Template Blueprint

Дата: 2026-03-25  
Статус: reusable template blueprint

## 1. Template goal

Каждая service page должна быть одновременно:

- SEO landing page;
- trust page;
- conversion entry page.

## 2. Block order

### Block 1. Localized hero

- clear service H1
- short practical intro
- relevant image or image-supported panel
- CTA to conversation

### Block 2. Typical scenarios

Purpose:

- показать, с какими задачами люди сюда приходят

Content:

- short scenario items
- practical examples

### Block 3. Scope of work

Purpose:

- объяснить, что именно выполняется

Content:

- service scope
- work categories
- relevant details

### Block 4. How work runs

Purpose:

- показать порядок и процесс

Content:

- step-based explanation
- communication logic
- object assessment

### Block 5. Pricing factors

Purpose:

- объяснить, от чего зависит стоимость

Content:

- object condition
- scope complexity
- materials
- access conditions
- timing

### Block 6. Proof/projects

Purpose:

- дать реальные фото и кейсы

Content:

- selected case cards
- before/after logic where possible

### Block 7. FAQ

- crawlable
- practical
- service-specific

### Block 8. Final CTA

- human conversation prompt
- phone
- concise next step explanation

## 3. Service-specific mapping

### Adaptacije kuća i stanova

- strongest visual proof:
  `домик`, `Мансарда до:после`, `Гипсокартон`

### Adaptacije kupaonica

- strongest visual proof:
  `Санузлы`, `работа с плиткой`

### Fasade

- strongest visual proof:
  exterior-oriented photos, selected `Hero` images, `наружные работы`

### Rekonstrukcije kuća

- strongest visual proof:
  `домик`, `Мансарда до:после`, `работа по дереву`

### Hidroizolacija

- strongest visual proof:
  `гидроизоляция`, supporting construction-process shots

## 4. UX rules

- no long walls of text
- no fake-estimate gimmicks
- no generic stock-service copy
- always end with a clear contact path
